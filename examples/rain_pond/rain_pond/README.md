# rain_pond

Intermediate Dog Paw teaching example.

Rain Pond demonstrates a slightly richer structure than `hello_dogpaw`:

- a controller that owns simulation state
- a service that listens to Dog Paw input
- keyboard fallback for local exploration
- a small settings surface for tuning the visual response

It is intentionally lighter than a full reference app. The goal is to show
service boundaries and multiple input shapes without overwhelming the reader
with every available control.
